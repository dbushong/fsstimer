FSS MultiTimer

✖
  • To set a timer, tap "Set" then enter the H:M:S, then tap the time
  • To stop a timer, tap the time again
  • After a timer has gone off, tap twice to reset to last setting
  • Set a timer to 00:00:00 to use as a count-up timer
  • Set a timer's name by tapping mic and saying the new name
  • If you say "reset" for the name, it will be reset to generic "Timer N", and just beep when it goes off.
  • Set time & name by hitting Set and saying something like "2 minutes 30 seconds" and optionally followed by "for <name>", e.g. "for turkey" to also set the timer name. You can say "reset" to default to the boring timer name + 00:00:00
